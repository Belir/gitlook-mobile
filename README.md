# gitlook-mobile

## Development HOWTO
1. Clone repo
2. Install yarn and react-native-cli globally
```
npm i yarn react-native-cli -g
```
3. Install dependencies
```
yarn
OR
yarn install
```
4. Run haul for specific platform or for all of them
```
yarn haul-(ios|android)
```
5. Run your app
```
yarn run-(ios|android)
```
